# WF-08 — YouTube Publish

## Purpose
Render nəticəsi YouTube-a yüklənir və video nəşr edilir.

## Inputs
- Run Contract
- WF-10 `resolved_config`

## Outputs
- Run Contract yenilənmiş manifest
- `youtube_video_id`

## Manifest update
- `publish.youtube_video_id` və `publish.publish_status` yenilənir.
- `timestamps.phase = published` və `timestamps.updated_at` yenilənir.
- `audit.change_log`-a “published” yazılır.

## Error handling
- Upload fail → retry (2 dəfə), sonra `upload_failed`.
- Metadata validation fail → `metadata_invalid`.
- Rate limit → `rate_limited` və exponential backoff.
- Xəta baş verərsə `FAILED_WF_08`, `timestamps.phase = failed_wf_08` və audit qeyd yazılır.

## Test cases
1. Upload uğurlu olduqda `youtube_video_id` dolur.
2. Title boş olduqda validation error verir.
3. Rate limit simulyasiyası backoff qaydasını göstərir.
