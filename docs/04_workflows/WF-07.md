# WF-07 — Render Orchestration

## Purpose
Render job yaradılır və nəticə izlənir.

## Inputs
- Run Contract
- WF-10 `resolved_config`

## Outputs
- Run Contract yenilənmiş manifest
- `render_id`

## Manifest update
- `render.render_id` və `render.render_status` yenilənir.
- `timestamps.phase = rendering` və `render_ready` mərhələləri yazılır.
- `audit.change_log`-a “render submitted” və “render completed” yazılır.

## Error handling
- Render submit fail → retry (2 dəfə), sonra `render_submit_failed`.
- Poll timeout → `render_timeout` və manual alert.
- Render result missing → `render_result_missing`.
- Xəta baş verərsə `FAILED_WF_07`, `timestamps.phase = failed_wf_07` və audit qeyd yazılır.

## Test cases
1. Render job yaradılır və `render_id` dolur.
2. Poll timeout ssenarisində status error olur.
3. Render nəticəsi yoxdursa error qaytarılır.
