# WF-10 — Get Config

## Purpose
Defaults + DB + runtime overrides ilə `resolved_config` qaytarır.

## Inputs
- Contract obyektindən `run_id`, `mode`, `scene_assets`, `voiceover_url_signed`, `render_id`, `youtube_video_id`, `manifest`
- `environment`
- `scope` və optional `channel_id`
- `runtime_overrides` (optional)

## Outputs
- `resolved_config`, `config_source`, `effective_config_summary`, `resolved_at`, `schema_version`
- Contract field-ləri dəyişmədən ötürülür

## Nodes plan
1. Trigger: HTTP/Internal Call
2. Load Defaults
3. Query Config DB (`ytb_settings`) — `continueOnFail` aktivdir
4. Apply Runtime Overrides
5. Normalize Output
6. Respond JSON

## Error handling
- DB unreachable → defaults ilə davam edir, `config_source=defaults`.
- Missing required keys → `config_incomplete`.
- Invalid environment → `invalid_environment`.

## Test cases
1. Defaults + DB + overrides birləşdirilir və `resolved_config` qaytarılır.
2. Missing key olduqda `config_incomplete` qaytarılır.
3. DB fail simulyasiyası defaults fallback qaytarır.
