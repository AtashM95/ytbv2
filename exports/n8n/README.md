# n8n Exports

Bu qovluq WF-01…WF-10 üçün n8n export JSON fayllarını saxlayacaq. Bu mərhələdə export faylları yaradılmır.

## Gələcək export qaydası
- Hər workflow ayrıca JSON olaraq export ediləcək.
- Fayl adlandırma qaydası: `WF-XX_<name>.json`
- Export edilərkən credentials məlumatları çıxarılmalıdır.
- Export sonrası checksum qeyd ediləcək.
